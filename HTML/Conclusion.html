Conclusion

L'analyse des données:

Après l'analyse des données fournies et manipulées, nous nous sommes rendu compte que l'environnement de l'entreprise est un milieu très complexe avec beaucoup d'informations à manipuler. La base SQL de KDou est bien organisée avec des relations claires entre les tables, ce qui facilite les requêtes et l’extraction d’informations pertinentes. Des indices et des jointures complexes peuvent ralentir certaines requêtes, suggérant une optimisation de la structure ou des indices à explorer.

Outils utilisés:

Durant cette SAE, nous avons utilisé de nombreux outils pour nous aider dans la représentation et l'utilisation des nombreuses données à notre disposition. En premier, nous avons visualisé les données à l'aide d'un QBE (Query By Exemple) qui nous permet de mieux voir les champs des tables pour pouvoir mieux s'en servir, il permet également de mieux voir les relations entre les tables ce qui est très utile pour la suite du travail pour faire les jointures des tables. En deuxième, nous avons utilisé les requêtes SQL qui sont des outils très puissants dans la manipulation des données fournies par la base de données. Nous avons créé de nombreuses requêtes et nous les avons interprétées. Durant la création des requêtes SQL, nous avons également utilisé le QBE qui nous permet de visualiser avec une vue plus globale la problématique présentée et une possible solution apportée.

Rédaction du rapport

Pour chaque page, un fichier HTML a été créé. La structure de chaque fichier suit un modèle de base, avec une balise "header" pour l'en-tête, un "nav" pour la navigation, des "section" pour le contenu principal, et un "footer" pour les informations de bas de page. Le code HTML est organisé de manière logique pour que chaque partie du contenu soit clairement définie. Navigation : La navigation entre les différentes pages est rendue possible par des liens "a" dans la barre de navigation, ce qui permet à l'utilisateur de passer d'une mission à l'autre.

Le CSS est utilisé pour styliser chaque page et garantir une présentation visuellement agréable et cohérente. Conception responsive : Les pages sont conçues pour être adaptatives (responsive), ce qui signifie qu'elles s'ajustent en fonction de la taille de l'écran. Styling cohérent : Les couleurs, polices et espacements ont été choisis pour créer un design harmonieux et professionnel. La page d'accueil, ainsi que chaque page des missions, partagent une charte graphique similaire.

Chaque page HTML contient un contenu détaillé sur les missions et le travail réalisé. Les sections ont été intégrées à partir des informations fournies par l'équipe, et une attention particulière a été portée à la clarté et à la précision des descriptions. Membres du groupe : La liste des membres du groupe a été présentée de manière simple sous forme de liste à puces. Description des missions : Chaque mission a été détaillée avec une explication des objectifs, des résultats obtenus et des actions entreprises.

Une fois le site développé, des tests ont été effectués pour vérifier son bon fonctionnement sur différents navigateurs (Chrome, Firefox, Edge) et sur différentes tailles d'écran. Le site a été optimisé pour garantir une navigation fluide et sans erreurs. Validation HTML et CSS : Le code a été validé à l'aide des outils de validation du W3C pour s'assurer qu'il respecte les normes web. Optimisation de la vitesse : Des ajustements ont été effectués pour améliorer la vitesse de chargement des pages, notamment en compressant les images et en optimisant le code CSS.

Le site web a été conçu pour offrir une présentation claire et structurée des missions réalisées, avec une navigation simple et efficace entre les différentes pages. Grâce à une bonne gestion des ressources HTML et CSS, le site offre une expérience utilisateur fluide, et les tests ont permis de garantir sa compatibilité avec différents navigateurs et appareils. Ce travail de développement a permis de mettre en pratique des compétences en conception web tout en offrant une interface visuellement agréable et fonctionnelle.
